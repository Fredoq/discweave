# Domain Knowledge

Core entities:

- [Artist](artist.md)
- [Release](release.md)
- [Track](track.md)
- [Medium](medium.md)
- [Owned Item](owned-item.md)
- [Credit](credit.md)
- [Label](label.md)

Core metadata includes artist, title, year or date, genre and user tags, label,
tracklist number, duration, file format, and media type.

The domain model must preserve relationships and roles, not just direct title
search.
