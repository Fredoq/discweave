# Human-Readable Export

DiscWeave must preserve the ability to export data in a human-readable format.

Exportability protects user trust and keeps the product aligned with its
personal archive purpose.

## Expectations

- Export should represent core entities and relationships clearly.
- Export should avoid opaque identifiers as the only readable connection between
  records.
- Export behavior should be tested when domain, storage, or serialization changes
  affect user data.
